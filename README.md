## Featured Projects

### 🔴 [TrustFlow — KOL Real-Time Analytics Platform](https://github.com/NATuan1208/kol-bigdata-realtime-analytics)
> Real-time trustworthiness scoring system for KOL (Key Opinion Leader) campaigns on social platforms

**Architecture:** Lambda Architecture — Speed Layer (Kafka + PySpark Streaming) + Batch Layer (PySpark + MinIO Lakehouse)

- Processed **372,806 social media records** with end-to-end pipeline latency < 2s
- Stacking Ensemble (LightGBM + XGBoost + Optuna) — **88.4% accuracy**, McNemar-validated
- Dual-role Redis caching layer — **94.7% cache hit rate**, reducing DB load significantly
- MLflow experiment tracking, Dockerized microservices, Prometheus + Grafana observability

`Python` `Apache Kafka` `PySpark` `MLflow` `Redis` `MinIO` `Docker` `LightGBM`
